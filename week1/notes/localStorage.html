Story Name:

My version of the story editor is for passwords. It has javascript and css in separate files, and stores password hints based on the site the user types. Originally I was going to place the passwords directly into localStorage, but then I read this article about why that is a terrible idea. https://dev.to/rdegges/please-stop-using-local-storage-1i04 That's why I decided to change it so that it just stores password clues/hints and not the actual password.
