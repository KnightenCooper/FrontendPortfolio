Notes:

Package managers: Keeps track of all of the external dependencies for our app.

Bundlers: Handle the compiling, transpiling, concatinating, minifying, and moving around of assets in our project.

Task managers: Keep track of what needs to be done and when.

node_modules: Where all the 3rd party tools are installed after you run "npm install".

devDependencies: Lists out all the tools necessary for a piece of software to run.

To run a program use "npm run start"
